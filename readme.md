# MAGI: A fun mixture of agents project
A min project that recreates the MAGI from the anime!
## Images
![Example #1](https://github.com/SammmyJr/Magi/blob/master/img/MaccasExample.jpg)
![Example #2](https://github.com/SammmyJr/Magi/blob/master/img/MathsExample.jpg)

## Usage
The GUI can be run by running the `gui.py` file after installing dependencies via `req.txt`.

## Notes
This uses 3 .guff files tailored to the Mistral 7B Q4 K M model, you will need to supply this file yourself and put it in the `models/balthasar` folder and create the models via Ollama!
